### Product Requirements - RoomLight Controller

| **ID** |           **Description**           |    **Type**    | **Priority** |
|:------:|:-----------------------------------:|:--------------:|:------------:|
|   01   |  RoomLight can connect to lighting  |   Functional   |     Must     |
|   02   | Supports multiple lighting settings |   Functional   |     Must     |
|   03   |   Any errors are directly reported  |   Functional   |     Must     |
|   04   |    RoomLight finds nearby devices   |   Functional   |     Must     |
|   05   | RoomLight can be used with no mouse |   Functional   |    Should    |
|   06   |  RoomLight has basic sound effects  |   Functional   |    Should    |
|   07   |   Application is for all platforms  | Non-Functional |     Must     |
|   08   |    RoomLight uses very little RAM   | Non-Functional |    Should    |
|   09   | Application can be installed easily | Non-Functional |    Should    |
|   10   |     Application has a simple UI     | Non-Functional |    Should    |
