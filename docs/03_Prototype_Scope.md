# Prototype Scope - RoomLight Controller
### Prototype Description
The RoomLight Controller prototype is a program that allows Bluetooth-compatible devices to connect to RoomLight lighting products.
It's mainly designed for hotel managers who want to make working with lights easier for employees and customers.
Having a functional prototype means that possible users can fidget with the settings and examine the quality-of-life improvements RoomLight can bring.

### Prototype Demonstration
The Prototype Description proves the following requirements:
- RoomLight can connect to lighting
- Supports multiple lighting settings
- Any errors are directly reported
- RoomLight finds nearby devices
These are proven through basic functionality in the application.
